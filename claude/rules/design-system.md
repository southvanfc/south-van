# Design System

## Brand Colours

| Token         | Hex       |
|---------------|-----------|
| Navy (primary) | `#013478` |
| Navy dark     | `#011f4a` |
| Navy deeper   | `#012860` |
| Gold (accent) | `#e8b430` |
| Gold light    | `#f5cc6a` |

## Typography

- Body: **Inter Variable**
- Accent headings: **Bebas Neue** (used for stats, hero labels)
- Secondary headings: **Oswald Variable**

## Responsive Breakpoints

Mobile-first design:

- `640px` — small tablet
- `960px` — desktop nav kicks in
- `1680px` — wide layouts
